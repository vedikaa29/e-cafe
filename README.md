A simple demo e-commerce web app for selling coffee and related products online.

## Tech Stack

- Node.js + Express (backend API & static file server)
- Vanilla HTML, CSS, JavaScript (frontend)
- JSON file as a simple product store

## Features

- Product listing (coffee, snacks, beans)
- Add to cart, update quantity, remove from cart
- Cart total calculation
- Cart persistence using `localStorage`
- Simple backend API at `/api/products`
- Ready to extend with Razorpay/Juspay for real payments

## Getting Started

```bash
git clone <your-repo-url>
cd e-cafe
npm install
npm run dev   # or: npm start
